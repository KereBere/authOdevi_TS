## This is a project simulates auth flow with JWT tokens

### Used technologies
- Express
- JWT tokens
- TypeScript
- Mongo db
- Heroku
- Validator
- HTML-CSS
- Ejs Template Engine
- MVC Architechture

## Installation

Clone the project to your local repository
```javascript
git clone https://github.com/KereBere/authOdevi_TS

```
Install the dependencies of the project

```
npm i
```
To run the project
```
npm run dev
```
